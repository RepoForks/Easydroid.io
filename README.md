# Easytouchforandroid.io
#模仿easytouch的安卓端app
##现有功能：
1.回到home主页<p>
2.打开系统自带拨号<p>
3.信息<p>
4.相机界面<p>
5.开启或关闭蓝牙<p>
6.移动网络开关<p>
7.wifi开关<p>
8.铃声开关<p>
9.输入关键字选择搜索引擎进行网络搜索<p>
