# Easytouchforandroid.io
一个模仿easytouch的安卓端app，主要包含一些快捷操作功能。
现有功能：回到home主页、打开系统自带拨号、信息、相机界面、开启或关闭蓝牙、移动网络、wifi、铃声，以及输入关键字选择搜索引擎搜索
