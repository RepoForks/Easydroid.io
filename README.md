# Easytouchforandroid.io
#模仿easytouch的安卓端app
##现有功能：
1.回到home主页
2.打开系统自带拨号
3.信息
4.相机界面
5.开启或关闭蓝牙
6.移动网络
7.wifi
8.铃声
9.输入关键字选择搜索引擎进行网络搜索
